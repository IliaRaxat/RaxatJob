# Фиксированные размеры карточек вакансий

## Описание изменений

Реализованы фиксированные размеры для всех карточек вакансий, чтобы они не изменялись из-за разного количества текста.

## Изменения в файлах

### 1. `src/app/jobs/jobs.module.css`
- **Фиксированная высота**: 280px для основных карточек вакансий
- **Flexbox layout**: `display: flex; flex-direction: column`
- **Ограничение описания**: до 3 строк с многоточием
- **Фиксированные элементы**: заголовок, теги, футер не сжимаются
- **Автоматическое позиционирование**: футер прижимается к низу

### 2. `src/app/jobs/my/my-jobs.module.css`
- **Фиксированная высота**: 200px для карточек "Мои вакансии"
- **Аналогичная структура**: flexbox с фиксированными элементами
- **Компактный дизайн**: подходит для списка собственных вакансий

### 3. `src/app/companies/companies.module.css`
- **Фиксированная высота**: 320px для карточек в разделе компаний
- **Расширенное описание**: до 4 строк для более подробной информации
- **Ограничение тегов**: максимальная высота 60px

## Ключевые особенности

### Структура карточки
```css
.jobCard {
  height: 280px; /* Фиксированная высота */
  display: flex;
  flex-direction: column;
  overflow: hidden;
}
```

### Элементы карточки
- **Заголовок**: `flex-shrink: 0` - не сжимается
- **Описание**: `flex: 1` - занимает оставшееся место
- **Теги**: `max-height: 60px` - ограниченная высота
- **Футер**: `margin-top: auto` - прижимается к низу

### Обрезка текста
```css
.jobDescription {
  display: -webkit-box;
  -webkit-line-clamp: 3; /* Ограничение строк */
  -webkit-box-orient: vertical;
  text-overflow: ellipsis;
  overflow: hidden;
}
```

## Результат

- ✅ Все карточки имеют одинаковую высоту
- ✅ Текст обрезается с многоточием при превышении лимита
- ✅ Футер всегда находится внизу карточки
- ✅ Адаптивный дизайн сохранен
- ✅ Hover-эффекты работают корректно

## Адаптивность

Изменения применяются ко всем размерам экрана:
- Десктоп: полная высота карточек
- Планшет: сохранение пропорций
- Мобильные: адаптивные отступы и размеры

## Совместимость

- ✅ Все современные браузеры
- ✅ Webkit-префиксы для line-clamp
- ✅ Fallback для старых браузеров
- ✅ Сохранение существующего функционала
