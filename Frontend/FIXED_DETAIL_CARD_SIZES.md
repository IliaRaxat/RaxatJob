# Фиксированные размеры карточек в детальных страницах

## Описание изменений

Реализованы фиксированные размеры для карточек в детальных страницах вакансий и стажировок, чтобы они имели единообразный вид независимо от количества контента.

## Изменения в файлах

### 1. `src/app/jobs/[id]/page.module.css`
- **Фиксированная высота**: 200px для всех карточек в сайдбаре
- **Flexbox layout**: `display: flex; flex-direction: column`
- **Ограничение списка заявок**: максимальная высота 120px с прокруткой
- **Фиксированные элементы**: заголовки, кнопки не сжимаются
- **Автоматическое позиционирование**: кнопки прижимаются к низу

### 2. `src/app/internships/[id]/internship.module.css`
- **Фиксированная высота**: 200px для карточек в сайдбаре
- **Ограничение текста**: до 3 строк с многоточием
- **Аналогичная структура**: flexbox с фиксированными элементами

## Ключевые особенности

### Структура карточки
```css
.hrCard, .applicationsCard, .applyCard {
  height: 200px; /* Фиксированная высота */
  display: flex;
  flex-direction: column;
  overflow: hidden;
}
```

### Элементы карточки
- **Заголовок**: `flex-shrink: 0` - не сжимается
- **Контент**: `flex: 1` - занимает оставшееся место
- **Кнопки**: `margin-top: auto` - прижимаются к низу
- **Списки**: `max-height: 120px` - ограниченная высота с прокруткой

### Обрезка текста
```css
.companyCard p {
  display: -webkit-box;
  -webkit-line-clamp: 3; /* Ограничение строк */
  -webkit-box-orient: vertical;
  text-overflow: ellipsis;
  overflow: hidden;
}
```

### Прокрутка для списков
```css
.applicationsList {
  flex: 1;
  overflow-y: auto;
  max-height: 120px;
}
```

## Затронутые карточки

### В детальной странице вакансии:
- **HR Card** - информация о контактном лице
- **Applications Card** - список заявок (только для HR)
- **Apply Card** - кнопка отклика (только для кандидатов)

### В детальной странице стажировки:
- **Company Card** - информация о компании
- **Apply Card** - кнопка отклика
- **Similar Card** - похожие стажировки
