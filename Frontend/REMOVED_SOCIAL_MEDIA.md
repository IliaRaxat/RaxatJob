# Удаление ссылок на социальные сети

## Описание изменений

Полностью удалены все ссылки на социальные сети из проекта, включая поля в профилях пользователей и упоминания в интерфейсе.

## Изменения в файлах

### 1. `src/lib/api/authApi.ts`
- **Удалены типы**: `socialMedia` из всех интерфейсов профилей
- **Очищены DTO**: убраны поля Facebook, Instagram, LinkedIn, Twitter
- **Обновлены интерфейсы**: `UniversityProfile`, `CreateUniversityProfileDto`, `UpdateUniversityProfileDto`

### 2. `src/app/profile/page.tsx`
- **Удален тип**: `socialMedia` из локального интерфейса
- **Убрана инициализация**: поля социальных сетей в форме

- **Обновлена статистика**: счетчик социальных сетей установлен в 0

### 3. `src/app/profile/profile.module.css`
- **Удалены стили**: `.socialMediaSection`, `.socialMediaGrid`
- **Очищены медиа-запросы**: убраны стили для мобильных устройств

### 4. `src/app/Components/JobApplicationForm.tsx`
- **Обновлен текст**: убрано упоминание LinkedIn из подсказки
